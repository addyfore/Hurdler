# Hurdler
Just a little jumper game I made from a tutorial program! Enjoy.
# https://liluo.io/spyoon/hurdler

# INSTRUCTIONS
LEFT - Move left
RIGHT - Move right
UP - Move forward
SPACE - Jump
ESC - Pause the game
