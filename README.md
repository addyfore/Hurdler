# Hurdler
Just a little jumper game I made from a tutorial program! Enjoy.
# https://liluo.io/spyoon/hurdler

# INSTRUCTIONS
LEFT - Move left <br>
RIGHT - Move right <br>
UP - Move forward <br>
SPACE - Jump <br>
ESC - Pause the game
